Google Apps Script Setup
Step-by-step setup:
  1. Go to script.google.com and create a New Project
  2. Paste the contents of gas/trigger.gs from this project
  3. Click Deploy → New Deployment → Web App
  4. Set "Execute as: Me" and "Access: Anyone"
  5. Copy the Web App URL and paste it in Settings above
  6. In the Apps Script editor, go to Triggers and add a time-based trigger to run checkAndTrigger every 5 or 10 minutes
⚠️ The GAS script reads your schedule from the BulkWA webhook endpoint and triggers GitHub Actions accordingly.
